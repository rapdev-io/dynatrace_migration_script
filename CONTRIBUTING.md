# Contributing

Issues and pull requests are welcome.

Before opening a pull request:

1. Run the test suite:

```bash
python3 -m unittest discover -s tests -v
```

2. Keep changes focused on the stated workflow:
- inventory normalization
- parity assessment
- dashboard recommendation and menu generation
- Terraform planning and deployment support

3. Avoid mixing unrelated refactors into the same change.

4. Update `README.md` when CLI behavior, auth requirements, or outputs change.

## Development Notes

- The repository supports both local JSON inputs and live API-backed workflows.
- The current implementation intentionally separates planning from full query translation.
- Terraform resources generated by this repo should be reviewed before apply.
